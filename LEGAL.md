# 1. Nature of This Tool  
**The Inner Path Map** is a free, bilingual (EN/中文) meditation-oriented tool based on **Buddhist psychology**.  
It provides general insights into:

- Practitioner tendencies  
- Disposition patterns  
- Entry meditation methods  

This system is built using frameworks such as the Five Spiritual Faculties（五根五力）and Seven Factors of Awakening（七覺支）from classical Buddhist psychology.

**It is not a religious teaching, ritual, divination, or metaphysical service.**  
It does not promote any specific lineage, sect, or belief system.

The content is for **general well-being, mindfulness support, and educational purposes only**.
